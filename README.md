# Lista de Tarefas em Python

Projeto simples de lista de tarefas desenvolvido em Python com interface gráfica usando Tkinter.

## Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Salvar tarefas automaticamente

## Tecnologias

- Python
- Tkinter

## Como executar

1. Baixe o projeto
2. Abra a pasta no terminal
3. Execute:

python main.py
