# Lista de Tarefas em Python

Sistema de lista de tarefas desenvolvido em Python com interface gráfica usando Tkinter.

## Funcionalidades

- Adicionar tarefas
- Mostrar tarefas na lista

## Tecnologias

- Python
- Tkinter

## Como executar

1. Baixe o projeto
2. Abra a pasta
3. Execute o arquivo:

python main.py
